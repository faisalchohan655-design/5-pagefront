# 🚀 LeadAI - AI-Powered Lead Generation Platform

## 📌 Overview
LeadAI is a powerful, AI-driven lead generation platform with 5 integrated pages for finding, managing, and converting leads.

## 🎯 Features

### 1. AI Dashboard
- Real-time analytics and stats
- AI-powered lead scoring
- Smart filtering and search
- Excel/CSV export
- Bulk delete

### 2. Smart Lead Finder
- Multi-platform search (Google Maps, LinkedIn, Facebook)
- AI enrichment
- Smart filters
- Save leads directly

### 3. Campaign Outreach
- WhatsApp and Email campaigns
- AI-generated templates
- Bulk messaging
- Campaign analytics

### 4. Lead Manager
- List and Kanban views
- AI scoring
- Smart pipeline
- Bulk operations

### 5. Email Extractor
- Single URL extraction
- Bulk URL extraction
- Email validation
- Phone number extraction
- Social profile detection

## 🛠️ Tech Stack
- React 18
- Tailwind CSS
- Lucide Icons
- XLSX (Excel/CSV export)
- Axios
- React Router
